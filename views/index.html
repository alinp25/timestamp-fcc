API Project: Timestamp Microservice

    User Stories:
  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date.
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it doesn't contain a date or a Unix timestamp, it returns null for those properties.

Example usage:

  • https://topaz-sea.glitch.me/December 15, 2015
  • https://topaz-sea.glitch.me/December%2015,%202015
  • https://topaz-sea.glitch.me/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
